Fortran(1957) – Один из первых высокоуровневых языков программирования, созданный для инженерных расчетов.
На Fortran до сих пор пишутся программы, особенно в области численных вычислений и суперкомпьютеров.
*****
Lisp (1958) – Язык, разработанный для работы с искусственным интеллектом, отличающийся обработкой списков.
Это один из старейших языков, который до сих пор используется, особенно в области искусственного интеллекта.
*****
COBOL (1959) – Язык для разработки бизнес-приложений и финансовых систем, особенно в банковском секторе.
Примерно 80% транзакций в финансовых системах мира всё ещё выполняются с использованием COBOL.
*****
C (1972) – Очень популярный язык общего назначения, оказавший огромное влияние на последующие языки программирования.
Большинство современных языков программирования, включая Python, Perl и Java, взяли многое от C.
*****
C++ (1985) – Надстройка над C, включающая объектно-ориентированное программирование.
Один из первых языков, который широко использовался в разработке видеоигр и программного обеспечения с интенсивным использованием графики.
*****
Python (1991) – Высокоуровневый язык программирования общего назначения с акцентом на простоту и читабельностьPython назван в честь комедийной группы Monty Python, а не в честь змеи.
*****
Java (1995) – Кроссплатформенный язык, широко используемый в разработке корпоративных приложений и веб-сервисов.
Java был изначально разработан как язык для программирования бытовой электроники, но позже стал основным языком для веб-приложений.
*****
JavaScript (1995) – Язык сценариев, основной в разработке веб-приложений для клиентской стороны.
JavaScript был разработан всего за 10 дней Бренданом Эйхом.
