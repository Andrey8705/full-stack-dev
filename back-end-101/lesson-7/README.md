# To-Do List Manager

## Описание

Простое консольное приложение для управления списком задач.

## Установка

1. Убедитесь, что у вас установлен Python 3.7 или новее.
2. Клонируйте репозиторий или скопируйте файлы проекта.
3. Установите зависимости (если есть).
